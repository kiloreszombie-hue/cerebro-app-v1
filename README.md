# cerebro-app-v1
Tu segundo cerebro - Aplicación para organizar tu vida

## Ejecutar la app
1. Abre una terminal en este proyecto.
2. Ejecuta:

```bash
npm start
```

3. Abre en tu navegador:

```text
http://localhost:3000
```

## Crear carpeta descargable (y ZIP)
Si quieres compartirla o moverla fácil, genera una carpeta lista para usar:

```bash
npm run package
```

Esto crea:
- `dist/cerebro-app/` (carpeta completa)
- `dist/cerebro-app.zip` (si el comando `zip` está disponible en tu sistema)

Luego puedes copiar/descargar esa carpeta o ZIP y ejecutar la app en otra máquina con:

```bash
npm start
```

dentro de la carpeta extraída/copiada.
